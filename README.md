RunMQ Dashboard is a web-based application that provides realtime monitoring and management capabilities for RunMQ. 
It offers an intuitive interface to visualize performance, queues, DLQs and manage various aspects of RunMQ.

## Features
- **Real-time Monitoring**: View live statistics on message throughput, queue depth, and broker
- **Queue Management**: Create, delete, and configure queues directly from the dashboard
- **Queue Configuration**: Updating attempts retry delay for queues
- **DLQ Management** Reprocessing messages from dead-letter queues
- and more. 

Work is still in progress, so stay tuned.
